# 股票查詢網站

## 作品介紹[影片](https://www.youtube.com/watch?v=lOTrmN7OUZI)

## 作品內容
此系統基於django架構，以python作為後端程式，使用twstock模組即時抓取台灣股票的即時資訊，並建立sqlite資料庫，存放近一年的歷史資料。\
前端使用html、css與js建立網頁，於首頁可選擇查詢即時資訊或歷史資訊，搜尋後即跳至該頁面。
